NibeWP (14107)

Der Baustein dient zur Kommunikation mit einer Nibe Wärmepumpe. Der Baustein nutzt hierbei das NibeUPD Gateway

Vorbereitung

  1. NIBE UDP Gateway läuft, s. z.B. das NIBE UDP Gateway auf Prodino MKR
  2. Mit NIBE Modbus Manager:
    1. In NIBE Modbus Manager Modell der Wärmepumpe auswählen
    2. Mit File - Export, Datei export.csv generieren
  3. export.csv mittels hsupload auf den HS laden:
    1. hsupload ist normalerweise unter folgendem Pfad zu finden:
      "c:\users\user_name\Documents\Gira\HS+FS Experte 4.11\Projekte\project_name\project_name\hsupload"
    2. In hsupload ddie UNterverzeichnisse \_logic\14107 anlegen
    3. export.csv in das neue Unterverzeichnis kopieren. Der Komplette Pfad zur Datei könnte dann so aussehen:
      "c:\users\user_name\Documents\Gira\HS+FS Experte 4.11\Projekte\project_name\project_name\hsupload\_logic\14107\export.csv"
    4. In Experte, ggf. Haken unter Projekt - Oberfläche für den Ordner setzen.
    5. Projekt übertragen.
